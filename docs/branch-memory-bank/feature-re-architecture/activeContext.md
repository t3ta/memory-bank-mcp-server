# アクティブコンテキスト

## 現在の作業内容

タグ検索、タグインデックス更新、最近のブランチ取得ユースケースを実装しました。
## 最近の変更点

- タグによるドキュメント検索のためのSearchDocumentsByTagsUseCaseの実装
- タグインデックス更新のためのUpdateTagIndexUseCaseの実装
- 最近のブランチの取得のためのGetRecentBranchesUseCaseの実装
- GlobalControllerとBranchControllerを新しいユースケースを使うように更新
- DIコンテナに新しいユースケースを登録
## アクティブな決定事項

- タグ検索はブランチにもグローバルにも同じユースケースを利用し、パラメータで切り替えるように実装
- タグインデックスは実際のインデックス対象となるドキュメントをスキャンしてタグを収集するように実装
- 各コントローラーをインターフェースに従って拡張し、新しいユースケースを使用するよう実装
