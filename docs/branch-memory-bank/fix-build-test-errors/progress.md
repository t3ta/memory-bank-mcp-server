1. **インポートパスの修正**:
   - ESモジュール方式に対応するため、インポートパスに.js拡張子を追加

2. **テストの問題修正**:
   - 未使用変数/インポートの削除
   - any型エラーの解決と型安全なコード作成
   - モックオブジェクトの型付け修正
